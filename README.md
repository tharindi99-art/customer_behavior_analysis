📝Overview

This project demonstrates an end-to-end data analytics workflow, starting from raw dataset processing to business insights visualization.
The project includes:
    • Loading and processing data using Python
    • Performing Exploratory Data Analysis(EDA)
    • Cleaning and transforming data
    • Running analytical queries using PostgreSQL
    • Building an interactive Power BI Dashboard
    • Creating a structured analytical report
    • Designing a professional presentation(PPT) using Gamma AI
The goal of this project is to extract meaningful insights from data and present them in a clear, decision-oriented format.
